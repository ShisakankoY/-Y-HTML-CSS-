エディタ(コードを打つところ)のダウンロード

VS Code（ブイエスコード）のインストール手順

1. 【公式サイトにアクセス】

下のリンクをクリック：

👉 https://code.visualstudio.com/

2. 【ダウンロード】

自分のパソコンの種類に合ったボタンをクリック：

例）Windowsの人 →「Download for Windows」

3. 【インストーラーを開く】

ダウンロードが終わったら、画面の左下（またはダウンロードフォルダ）に出てくる

VSCodeUserSetup.exe をダブルクリック。

4. 【インストールを進める】

「同意します」にチェック ✅

「次へ」を何回かクリック → 「インストール」でOKです。

（細かい設定はそのままでOK）

5. 【起動する】

インストール完了後、「Visual Studio Code を起動」にチェックを入れて「完了」をクリック！

ファイルの名称設定

ファイルの名前は何でも良いです。

しかし!拡張子(名前の後のドットに続く文字)は決まっています。

htmlファイル→.html
cssファイル→.css
(Java Script→.js)
