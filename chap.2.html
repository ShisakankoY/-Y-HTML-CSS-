CSSとは？

CSSとは、スタイルシート言語と呼ばれる、

HTMLを装飾(色付け・文字の並び順の変更など)する言語(ツール)です。

CSSは、class属性・id属性で紐づけるか、タグ名を記入することで、反映させることができます。

それでは、詳しく見ていきましょう！

CSSの書き方

タグ名での指定

          body {
            background-color: #f0f0f0;
          }
class名での指定

          .container {
            display: flex;
            justify-content: center;
            align-items: center;
            margin: 100px;
          }
id名での指定

          #highlight {
            color: white;
            background-color: crimson;
            padding: 10px;
            border-radius: 5px;
          }
総括・書き方まとめ

          紐付ける要素名 {
            指示内容
          }

このように、タグ名で指定の時は波括弧の前にタグ名を、

class名での指定は.class名(ドットを入れる！)を波括弧の前に、

id名の場合は、#id名(#を入れる！)を波括弧の前に書きます。

また、波括弧の中に、指示を書き込みます。

そして、指示を書くときは、波括弧の中に、「プロパティ＋コロン(:)＋値＋セミコロン(;)」の順で書きます。


指示解説(抜粋)

※何を設定するかを表す項目名をプロパティと言います。どういう状態にするか、を伝えるための具体的な指定を値と言います。

background-color: #f0f0f0;

これは、背景を決めるCSSです。値には、カラーコードを入れ、それで色を指定します。


display: flex;

これは、要素(Webサイトの一つ一つのパーツのようなもの(文章など))を柔軟に配置するためのCSSです。

この場合、横並びになります。

justify-content: center;

これは、Flexboxを使う時(display: flex;が良い例)、横方向に中央揃えするのに使います。

align-items: center;

これは、Flexboxを使う時,縦方向に中央揃えにするときに使います。

margin: 100px;とpadding: 10px;

余白を設定しています。marginが要素の外側の余白、paddingが内側の余白を表します。
