チャプター1 HTMLとは？~骨組みを知ろう~

HTMLとは？

HTMLとは、マークアップ言語と呼ばれる、

Webサイトの骨組みをつくる言語(ツールと言えばいいかな？)です。

なお、Webサイトの装飾(色付け・文章などの並べ方の変更など)はCSSを用います。

HTMLは、「タグ」と呼ばれるコードを組み合わせて構成されています。

タグは不等号で囲まれています。また、コードは全て半角で入力します。

そして、タグには「属性」と呼ばれる、+αで情報を追加できるようなコードを組み合わせることができます。

※個人的に最重要属性

class, id→CSSで装飾するときに必要な合言葉。これに適用することで、装飾ができる。なお、タグ名でも指定できる。

※idは一つの合言葉を持ったものは、一回しか使えない。また、ナビバーでスクロールするところを指定するときにも使える。

基本的なタグ

見出し

              <h1><h2><h3><h4><h5><h6>例：<h1>こんにちは</h1>

見出しに使います。1~6まであり、1の方が大きいです。h1は基本的にページで一回だけ使います。

タグは基本的に、例のようにスラッシュも含めて囲み、タグの中に出力する文字を記入します。

文章

              <p>テキスト</p>

文章に使います。

リンクを貼る

              <a href="ここにリンクを挿入"></a>

リンクを貼るときに使います。hrefは属性の一つで、その後にリンクを挿入します。

また、属性に値(ここではリンクのこと)を入れるときは、ダブルクォーテーション("")で囲みます。

画像を貼る

              <img src="ここにリンクを挿入" alt="ここに読み上げや画像が表示されないときに代わりに表示するために使う捕捉情報を入れる">

画像を貼るときに使います。src属性にリンク、alt属性に読み上げや画像が表示されないときに代わりに表示するために使う捕捉情報を入れます。

なお、imgタグは、最後にタグで囲う必要がありません。

箇条書きリスト

              <ul>
                <li>テキスト</li>
                <li>テキスト</li>
                <li>テキスト</li>
              </ul>

箇条書きリストは、全体をulで囲い、各要素をliで囲みます。

なお、順序付きリスト(番号など)を作る際は、ulの代わりにolを使います。

骨組みを作るタグ

骨組み

              <html></html><head></head><body></body><header></header><footer></footer>

htmlタグで全体を囲い、その直後で、出力はしないプログラム上で必要な情報をまとめるのにheadタグを使います。

出力する本体をbodyで囲い、ナビバー(ページ上部の、ロゴや、ページ内で他のところに飛んだりできるところ/当ページの上部黒いところ)などを作るのがheaderタグです。

そして、著作権表記などがされてあるページの一番下のところをfooterで囲みます。

その他、主にbodyで使う区分するタグ

              <div></div><span></span><main></main><section></section><article></article>

divは、class属性などで指定し、装飾したいところをまとめるときに使います。結果、divで囲まれたところ全体にCSSが適用されます。また、divは前後に改行が入ります。

spanは、大体divと似ているのですが、前後に改行が入りません。なので、文中など限定的な装飾に便利です。厳密な違いは、検索していただいた方がいいかと思います。

mainは、コンテンツ内容において、コード全体の中のメインパートを囲みます。

sectionはmainの中の内容分けできる範囲を区分し囲みます。

articleはsectionの中でも独立したコンテンツ(記事など)を囲みます。

骨組みの例


            <!DOCTYPE html>
            <html lang="ja">
            <head>
              <meta charset="UTF-8">
              <meta name="viewport" content="width=device-width, initial-scale=1.0">
              <title>HTMLサンプルページ</title>
            </head>
            <body>
            
              <!-- メイン見出し -->
              <h1>こんにちは、世界！</h1>
            
              <!-- 概要説明 -->
              <p>これはHTMLの基本構造を学ぶためのサンプルページです。</p>
            
              <!-- 箇条書きリスト -->
              <ul>
                <li>HTMLとは？</li>
                <li>タグの使い方</li>
                <li>ページ構造の作り方</li>
              </ul>
            
              <!-- リンクと画像 -->
              <p>詳細は <a href="https://developer.mozilla.org/ja/docs/Web/HTML">MDN Web Docs</a> をご覧ください。</p>
              <img src="sample.jpg" alt="サンプル画像" width="300">
            
              <!-- フッター -->
              <footer>
                <p>© 2025 サンプルサイト</p>
              </footer>
            
            </body>
            </html>
解説

全体をhtmlタグで囲い、データをheadタグで囲んでいます。

bodyで囲まれた中に、見出し、文章、リスト、リンクや画像が記述されています。(基本的にタグはタグの中に入れることができます。)

bodyの最後に、footerでフッターを作っています。

また、画像を貼るimgタグの中にある、width="300"は、画像の幅を指定しています。

headタグに書くこと


            <!DOCTYPE html>
            <html lang="ja">
            <head>
              <meta charset="utf-8">
              <title>タイトル</title>
              <meta name="description" content="説明">
              <meta name="viewport" content="width=device-width, initial-scale=1">
              <link rel="icon" href="アイコンの画像リンク">
              <link rel="stylesheet" href="https://unpkg.com/ress/dist/ress.min.css">
              <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css"
                  rel="stylesheet">
              <link rel="stylesheet" href="カスタムCSSファイル名">
            </head>
解説

まず、コードの一番上に、ドキュタイプ宣言を必ずします。

次に、htmlタグを記入、言語を設定します。(ja=日本語)

そして、headタグで囲みます。

その中に、文字コードを設定し、サイトのタイトル(タブに出てくる名前)を記入します。

そうしたら、検索されたときに出す説明を記述、その後、meta name="viewport" content="width=device-width, initial-scale=1"を記入します。

その後、アイコン(PNG形式推奨)と外部CSSリンク・BootstrapのCSSリンクを貼ります。

最後に、カスタムCSSファイルのリンクを貼ります。

これが、個人的にまとまったと感じるheadタグです。

ここに必要に応じて、コードを消したり、追加します。

※BootStrapについて

Bootstrapは、フレームワークといって、CSSで指定しなくても、決められたclass属性をつけるだけで装飾をしてくれる、便利なツールです。

使用するには、head内のリンクの他、bodyタグの下部に、Bootstrap用のJava Scriptのリンクを貼ります。(同じscriptタグで、自分のJava Scriptを読み込ませることも可能です。)

Bootstrap jsリンク

            <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
